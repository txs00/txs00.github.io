react三大核心属性-ref

三种ref使用方式：
    1.通过绑定ref，值为字符串，获取dom元素,this.refs.input
    2.通过ref绑定一个回调函数形式，形参就是当前的dom对象
    3.通过React.createRef()创建一个ref对象，进行ref绑定
    示例：ref={this.refDom}

受控组件和非受控组件

使用state进行控制的组件
不使用state控制的组件

高阶函数

1.一种是以函数形式作为参数的函数，叫做高阶   列举：Promise(resolve,reject)  map(()=>{})
2.一种以函数形式返回的函数，叫高阶

函数柯里化

let add = function(a,b,c) {
    return a + b + c
}
console.log(add(1,2,3))

let add2 = function(a) {
    return (b)=>{
        return (c)=>{
           return a+b+c
        }
    }
}
console.log(add2(1)(2)(3))
理解为函数的拆分细化，最后统一处理逻辑
